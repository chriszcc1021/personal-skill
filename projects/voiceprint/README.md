# voiceprint · 声纹歌手匹配

测你的唱歌音色像哪个歌手（华语内地 + 港澳台 + 日本 + 欧美，全球版 Deep Voice）。

## 一句话
用户清唱 15-30 秒 → 提声纹向量 + 音域分析 → 跟 50 位歌手声纹库比对 → 输出音色/音域双维度 Top 匹配 + 相似度分数，Spotify 风格结果页可分享。

## 🎯 已锁定决策（2026-07-19）
1. 歌手库：内地13 + 港澳台13 + 日本12 + 欧美12 = **50人**，不含韩国
2. 素材源：**唱歌干声**（official acapella / 歌曲Demucs分离），访谈说话声排除
3. 呈现风格：**Spotify 风**（深色 #121212 + 绿 #1DB954）
4. 建库用现成素材
5. **加音域检测**（第二维度，硬核可信，补声纹玄学）
6. 输入输出同源原则：库是唱歌声，用户也必须唱不能说

## 🏗️ 技术栈
- 后端：Python + FastAPI
- 声纹：Resemblyzer / ECAPA-TDNN（本地跑，开源免费，无API调用费）
- 音域：CREPE / pyin（音高检测）
- 人声分离（建库）：Demucs
- 前端：Web H5（Phase1）→ 微信小程序（Phase2）
- 部署：现成服务器 43.134.32.223 子路径
