# voiceprint PRD — 产品需求文档 (MVP)

## 1. 一句话定义
用户录一段自己的声音,测出音色最像哪些歌手,给出相似度分数,可分享。

## 2. 目标用户
KTV 爱好者、爱玩声音测试的年轻人、想知道自己"声线像谁"的普通用户。

## 3. 用户旅程 (MVP)
1. 打开网页 → 看到"测测你的声音像哪个歌手"引导
2. 点录音 → 提示"清唱或说话 15-30 秒,安静环境效果更好"
3. 录完 → 上传 → 等待(转圈,几秒)
4. 出结果:
   - Top3 匹配歌手 + 相似度百分比
   - 可选:你的声部/音域
   - 一张可分享的结果卡片
5. 可重录 / 可分享

## 4. 功能范围
### MVP 必做 (P0)
- [ ] 浏览器录音(15-30s,时长校验)
- [ ] 音频上传后端
- [ ] 声纹提取 + 歌手库比对
- [ ] 返回 Top3 匹配 + 分数
- [ ] 结果页展示
- [ ] 基础错误处理(录音太短/太吵/无人声)

### V1.1 可做 (P1)
- [ ] 结果分享卡片(生成图片)
- [ ] 音域/声部检测(第二维度,提升可信度)
- [ ] 歌手库扩到 100+

### 暂不做
- 用户账号系统
- 历史记录
- 付费功能
- 微信小程序(Phase 2)

## 5. 非功能需求
- 单次匹配延迟 < 3 秒
- 支持并发(先扛 50 并发,后续加机器)
- 移动端 H5 优先(大部分人手机玩)

## 6. 结果呈现原则
- 定位娱乐,不号称精准
- 分数别给太低(打击体验),做映射拉到"看着合理"区间
- 每个结果配"你和XX共同点:音色偏XX"这种解释,增强可信+趣味
