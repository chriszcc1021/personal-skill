# voiceprint 歌手库建库 SOP

> 全项目最大的体力活 + 质量风险。库不好,产品必崩。

## 1. 数据来源(按优先级)
1. **官方清唱/无伴奏版**:部分歌手有 acapella 版本,最干净,优先
2. **访谈/说话片段**:歌手说话音色最接近声纹模型训练分布,稳定
3. **歌曲 + Demucs 分离人声**:量大但有杂音,兜底手段

## 2. 单个歌手建库流程
1. 收集 3-5 段素材(清唱 or 说话,每段 ≥ 15s)
2. Demucs 分离(若带伴奏)→ 拿人声轨
3. ffmpeg 转 wav 16k mono + 去静音
4. 人工听一遍,剔除杂音重/混响大的片段(质量守门)
5. 每段提 embedding → 取平均 → 得该歌手最终声纹向量
6. 存入库(向量 + 元数据),**删掉原始音频**

## 4. 质量标准
- 每歌手 ≥ 3 段有效素材
- 单段信噪比达标(人工听感无明显杂音)
- 同歌手多段 embedding 内部余弦相似度 > 0.6(不然说明素材不纯/混入他人声)

## 5. 版权处理
- 库内**不存任何原始音频**,只存不可逆的向量
- 素材处理完即删
- 产品页标"仅供娱乐,声纹相似度参考"

## 6. 工期估算
- 熟练后单歌手约 20-30 分钟(收集+分离+清洗+提取)
- 26 人首批 ≈ 1.5-2 天纯建库
- 这是可并行/可外包的体力活,不卡技术
