# Design Inspiration Libs (3 个审美灵感库)

> 来源：小红书 vibecoding 笔记 (2026-05-20)。先把 3 个网站记牢，用法见下面。

## 使用心法

1. **先逛再写** — 写代码前用 5-10 分钟刷 reference，定 style direction (color / type / layout / motion)
2. **截图存档** — 看到喜欢的直接截图扔 `~/.openclaw/workspace/inspo/YYYY-MM/`，配 1 行笔记
3. **复刻 ≠ 抄** — 学结构 / 节奏 / 留白，颜色和字体替换成自己的
4. **避雷 AI 味** — 看到模板化的渐变 hero + 三栏 feature card 就警惕，从这 3 个库里挑 award 级案例做对照

---

## 1. Awwwards · 首推 🏆

- **URL**: https://www.awwwards.com
- **定位**: 全球最权威的网页设计奖项，提名/获奖站点质量极高
- **必看入口**:
  - Sites of the Day (SOTD)：https://www.awwwards.com/websites/
  - Sites of the Month / Year：年度精选
  - **Honors / Site of the Year**：顶级案例
  - Collections (Typography / Animation / Dark / Minimal …)：按风格筛
- **适合**: 找高端品牌站、agency 风、artistic 实验性案例
- **注意**: 偏炫技；做企业站参考要克制，别原样照抄重交互

## 2. Land-book · 第二个 🏘️

- **URL**: https://land-book.com
- **定位**: 海量 landing page 合集，专注「能落地的」商业站点
- **必看入口**:
  - Gallery：https://land-book.com/gallery
  - 按 **Industry**（SaaS / AI / Fintech / E-commerce …）筛
  - 按 **Style**（Minimal / Bold / Dark / Brutalist …）筛
  - 按 **Color** 找配色相近案例
- **适合**: SaaS / 工具类 landing、转化导向的产品站
- **注意**: 比 Awwwards 更实用，结构能直接学

## 3. Designeverywhere · 第三个 🎨

- **URL**: https://designeverywhere.cc
- **定位**: 综合设计灵感聚合（不止 Web，也含 UI 截图 / 海报 / 包装 / 品牌）
- **必看入口**: 首页瀑布流 + tag 筛
- **适合**: 跨媒介 mood board、找配色 / 字体 / 排版灵感
- **注意**: 不是纯 web 站点库，做品牌系统时更好用

## 工作流模板

```
任务：做一个 SaaS landing
1. Land-book → Industry=SaaS, Style=Minimal → 收 8-10 个截图
2. Awwwards → Collections=SaaS / Typography → 找 1-2 个 award 级做 north star
3. Designeverywhere → 搜配色关键词 → 锁定主色 + 辅助色
4. 整理成 mood board (Figma / Excalidraw)，标注「学哪部分」
5. 再开始写代码 / vibe with AI
```
