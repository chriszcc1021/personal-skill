3核心问题 · 部分致命

口碑层面

  • 缝合争议：被广泛指抄《致命公司》+《第五人格》，原创性饱受质疑
  • 主播负面事件拖累社区氛围（出轨 / 威胁前女友）
  • 未成年大额充值纠纷（2026-03 多家媒体曝光，退款难）
  • 公屏涉黄引流泛滥，监管不足

产品层面

  • 外挂泛滥：透视 / 自瞄横行，反外挂力度薄弱，举报形同虚设
  • 内容枯竭快：地图 / 怪物种类有限，重复感 1-2 月就堆积
  • 新手引导几乎缺失，老玩家垄断棺材点资源
  • 匹配 / 平衡差：单排崩、判定不准、穿模穿墙、移动端手感生涩
  • 付费虚高：8888 灵契 / 368 金砖单角色，性价比低

注：民间流传"70% 玩家退坑"，来源不严谨但社媒情绪普遍。Apple Store / TapTap 评分两极分化明显。

5海外出海判断

结论：审慎推荐出海，但巨人海外发行 infra 是最大变量（不像网易 / 腾讯 / 米哈游 / 莉莉丝有成熟体系）。产品本身稀缺性还在，能不能复制看两件事：补齐发行能力 + 把商业化重做一遍。

阶段市场目标判定
P0 · 0-3M港澳台 + 海外华人语言外推验证 + 海外节奏建立首选
P1 · 3-9M日本非华语接受度 + 整活直播红利强推
P2 · 9-18M北美 / 西欧（+ PC / Steam 同步）接 Lethal Company 基本盘条件许可
P3 · 看 P1 结果越南 / 泰国选择性流量市场扩量可选
不推荐首发韩国（infra 缺失）—缓
不推荐印尼 / 马来 / 中东 / 印度题材合规 / 结构不合略

核心商业化必须重做

国内靠中式时装 + 国漫 IP 联动（敦煌 / 傩戏 / 叶罗丽 / 景德镇）这套在海外不灵，需要重建本地 IP 联动池 + 重新定价（日本 ARPU 高但对抽卡敏感，时装定价偏高问题会暴露）。

∎牛子哥的一句话结论

国内 1000 万 DAU 是真的，但很大程度是「赛道空白 + 抄得对路 + 视频时代红利」三重叠加；海外能不能复制，看巨人能不能补齐发行能力，以及把商业化重做一遍。产品本身的稀缺性窗口还在，但窗口期不会太长 —— PEAK / R.E.P.O 这些西方鼻祖热度回落留出来的空当，可能 2026 年底就有人来填。
