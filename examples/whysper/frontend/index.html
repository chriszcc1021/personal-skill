把截图送进 Whysper

  1. 在「快捷指令」中新建动作：截屏。
  2. 添加「通过 URL 获取内容」，方法选 POST，请求体选表单。
  3. 表单字段添加 image（文件）、source（文本，shortcut）和 capture_mode（文本，auto）。
  4. 把下面这个地址填进 URL，然后在系统设置里把 Action Button 绑定到这个快捷指令。
  5. 需要自动加日历时，上传后每 5 秒查询一次 calendar-items，最多 3 次。
